Данный раздел описывает как интегрироваться с публичным API системы.

Для выполнения любых операций в системе необходимо сперва пройти регистрацию, а после этого получать токен доступа для обращения к любому доступуному методу, токен передается в заголовке accessToken.

Account

POST /api/Account/Authenticate

Аутентификация пользователя в системе.

 Migration Settings

POST api/Settings/SetSettings

Установить настройки пользователя.

Request body '{{ "{" }}' "sourceConnectionString": "string", "sourceDatabaseType": 0, "destinationConnectionString": "string", "destinationDatabaseType": 0 '{{ "}" }}'

GET /api/Settings/GetSettings

Получит настройки пользователя.

Данные получаются по переданному токену accessToken.
